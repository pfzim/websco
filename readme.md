# WebSCO - alternative web interface for System Center Orchestrator

## Installation

```
sudo apt-get install php php-mysql php-ldap php-curl php-xml
```


## Technical information

Bit `flags` in table `runbooks`

| Bits   | Description                               |
|--------|-------------------------------------------|
| 0x0001 | Deleted                                   |
| 0x0002 | Hide from list                            |
